# Code of Conduct

## Our Standard

ComfyStudio is committed to a welcoming, respectful, and constructive community.

Examples of behavior that help build that environment:

- Being respectful in disagreements.
- Giving actionable, good-faith feedback.
- Assuming positive intent while still being clear about problems.
- Helping newer contributors learn the project and release process.

Examples of unacceptable behavior:

- Harassment, intimidation, or discrimination.
- Personal attacks, insults, or hostile pile-ons.
- Publishing someone else's private information without permission.
- Bad-faith disruption of discussions, issues, reviews, or community spaces.

## Scope

This Code of Conduct applies in project spaces, issue trackers, pull requests, discussions, and any public spaces where a person is representing the project.

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code, issues, and other contributions that violate this Code of Conduct.

## Reporting

If you experience or witness unacceptable behavior, contact the project maintainer through a private channel rather than opening a public escalation thread with sensitive details.

## Consequences

Maintainers may take any action they deem appropriate, including warnings, temporary restrictions, or permanent removal from project spaces.
